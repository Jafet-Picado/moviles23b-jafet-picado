calculate() → dynamic
Solves the expression by using a preorder route on the binary tree and doing the operations between the child of each operation. Example: root = *, root.left = 2 and root.right = 3 The value returned would be 2 * 3 = 6 But, if there is an invalid input like x instead of * to multiplication, it would throw an ArgumentError exception
